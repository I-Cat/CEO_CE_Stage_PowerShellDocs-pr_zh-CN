# Microsoft.PowerShell.Diagnostics 模块
本部分包含**Microsoft.PowerShell.Diagnostics**模块的帮助主题。 **Microsoft.PowerShell.Diagnostics**模块包含管理事件日志、 性能计数器和跟踪 Windows 事件 (ETW) 的 cmdlet。 使用 Windows PowerShell 安装它。
