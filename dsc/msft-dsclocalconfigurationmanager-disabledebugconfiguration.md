# MSFT_DSCLocalConfigurationManager 类 DisableDebugConfiguration 方法

禁用 DSC 资源调试。

语法
------

```mof
uint32 DisableDebugConfiguration();
```

参数
----------

此方法没有任何参数。

## 返回值
------------

返回的零成功;否则将返回错误代码。

## 说明

这是静态的方法。

## 要求
------------
>**MOF:**DscCore.mof

>**Namespace**: Root\Microsoft\Windows\DesiredStateConfiguration
